4.7. Cayenne OSGI extension

Helps to bootstrap Cayenne in OSGi environment.

4.7.1. Maven

<dependency>
    <groupId>org.apache.cayenne</groupId>
    <artifactId>cayenne-osgi</artifactId>
    <version>5.0-M1</version>
</dependency>

4.7.2. Gradle

compile 'org.apache.cayenne:cayenne-osgi:5.0-M1'
